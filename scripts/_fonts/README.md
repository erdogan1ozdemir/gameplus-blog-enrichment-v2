# Fontlar (lisanslı — repoya konmaz)
Önizleme (`embed_fonts`) için şu dosyalar yerelde `scripts/_fonts/` altında bulunmalı:
- `new-science-semibold-ext.otf` (New Science SemiBold Extended — başlıklar)
- `reg.otf` (GreycliffCF Regular 400) · `med.woff2` (Medium 500) · `bold.woff2` (Bold 700)
Bunlar lisanslı olduğundan repoya dahil edilmez; ekipten temin edip bu klasöre kopyala.
Yoksa önizleme fallback fontla açılır (CMS gövdesi etkilenmez).
